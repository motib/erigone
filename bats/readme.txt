Download and open the archive erigone-N.zip, where N is the version
number. Open the archive in a clean directory.

The subdirectory docs contains a quick start guide and a user's guide.

The source code with its documentation is in erigone-source-N.zip.

The regresssion testing suite of Promela programs is in erigone-test-N.zip.

An archive of graphs of verifications is in vmc-examples-N.zip.
